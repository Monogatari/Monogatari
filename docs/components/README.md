# Components

Monogatari's interface is built from **components** — small, self-contained custom elements you can restyle, reconfigure, or replace. This section is the reference for the components you'll customize most often.

> [!TIP]
> New to components? Start with [the Components building block](../building-blocks/components/README.md)
> to learn how they work, then come back here for the reference on each one.

## Dialog & text

- [Text Box](text-box.md) — The main dialog box where characters speak.
- [Centered Dialog](centered-dialog.md) — A floating, centered dialog for dramatic moments.
- [Type Writer](type-writer.md) — The animated text engine, with inline effects.
- [Dialog Log](dialog-log.md) — A scrollable history of everything that's been said.

## Menus & navigation

- [Main Menu](main-menu.md) — The game's title-screen menu.
- [Quick Menu](quick-menu.md) — The in-game toolbar (save, load, settings…).
- [Choice Container](choice-container.md) — The buttons shown by the choice action.

## Screens

- [Loading Screen](loading-screen.md) — The asset-preloading progress screen.
- [Main Screen](main-screen.md) — The root game-screen container.
- [Settings Screen](settings-screen.md) — Player settings: language, text speed, volume…
- [Save Screen](save-screen.md) — Where players name and store saves.
- [Load Screen](load-screen.md) — Where players pick a save to load.
- [Gallery Screen](gallery-screen.md) — The unlockable image gallery.
- [Help Screen](help-screen.md) — In-game controls and shortcuts.
- [Credits Screen](credits.md) — Your game's credits.
- [Language Selection Screen](language-selection-screen.md) — The pre-game language picker.

# Chrome App

Google [discontinued apps on the Chrome Web Store](https://blog.chromium.org/2020/01/moving-forward-from-chrome-apps.html), so this is no longer a supported way to release your game.

Release it as a **[web](web.md)** game instead — it runs in Chrome and every other modern browser, and can still be [installed to a device](mobile.md) as a Progressive Web App.
